# Task 01 - House Price Prediction (Linear Regression)

## 📌 Objective
Predict house prices using:
- Living Area (GrLivArea)
- Number of Bedrooms
- Number of Bathrooms

## 📊 Dataset
Ames Housing Dataset (train.csv)

## ⚙️ Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📈 Model
- Linear Regression
- Train-Test Split (80/20)

## 📊 Evaluation
- MAE (Mean Absolute Error)
- R² Score
